# Seneca Playground

A very simple Seneca mesh micro-services setup to evaluate operational concepts.


### Install

  `git clone https://github.com/behrad/seneca-test.git`

  `yarn install`


### Run

```
port=40000 node base.js 
node service.js
node client.js
```
